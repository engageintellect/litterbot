# litterbot

## Description

This is a simple bot that will help you keep your project directory clean by moving files to a specified directory.

## Getting Started

### Server

```bash
$ cd server
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python main.py
```

### Client

```bash
$ cd client
$ pnpm i
$ pnpm run dev --host
```
