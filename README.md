# litterbot

## Description

An app to interact with a litter robot by whisker, using Sveltekit, Python, FastAPI, WebSockets, TailwindCSS, and more.

## Getting Started

### Server

```bash
cd server
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Client

```bash
cd client
pnpm i
pnpm run dev --host
```
